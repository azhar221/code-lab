# code-lab

A comprehensive coding practice repository with solutions to algorithmic problems, data structures, and coding challenges implemented in multiple programming languages.

## 📚 Contents

This repository contains solutions and practice code organized by programming language and topics:

### Languages
- **C** - Basic C programming concepts
- **C++** - Advanced C++ with data structures and algorithms
- **Java** - Object-oriented programming and DSA
- **Python** - Scripting and algorithm implementations
- **LeetCode** - Problem solutions from LeetCode

### Topics
- **Arrays** - Array manipulation, searching, sorting
- **Strings** - String processing and pattern matching
- **Recursion** - Recursive problem solving
- **LinkedList** - Linked list implementations and operations
- **Stack & Queue** - Stack and queue data structures
- **Binary Tree** - Tree traversal and operations
- **2D Arrays** - Matrix operations
- **Sorting** - Various sorting algorithms
- **Searching** - Search algorithms and techniques
- **OOP** - Object-oriented programming concepts
- **Patterns** - Design patterns and code patterns

## 🚀 Quick Start

### Prerequisites
- GCC/G++ (for C/C++)
- Java JDK (for Java)
- Python 3.x (for Python)

### Compilation & Execution

**C/C++:**
```bash
g++ filename.cpp -o output
./output
```

**Java:**
```bash
javac filename.java
java ClassName
```

**Python:**
```bash
python filename.py
```
